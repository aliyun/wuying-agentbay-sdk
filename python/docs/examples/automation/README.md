# 自动化功能示例

本示例展示了AgentBay SDK的自动化功能，包括命令执行、代码执行、UI自动化和工作流编排。

## 功能特性

- **命令执行**: 在云端执行Shell命令
- **代码执行**: 运行Python和JavaScript代码
- **UI自动化**: 屏幕截图、键盘输入、鼠标操作
- **工作流编排**: 多步骤自动化任务

## 运行示例

```bash
# 安装依赖
pip install wuying-agentbay-sdk

# 设置环境变量
export AGENTBAY_API_KEY="your-api-key"

# 运行示例
python main.py
```

### 1. 命令执行示例
- 基本系统命令执行
- 带超时的命令执行
- 错误处理

### 2. 代码执行示例

- JavaScript代码执行

- 键盘输入模拟
